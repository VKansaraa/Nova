# 🤖 Math Chatbot — Python Project

This is a simple command-line **chatbot written in Python** that can both **solve math problems** and **chat with the user** in a friendly way. It includes a range of useful math-solving tools and aims to be both educational and interactive.

---

## 🧠 Features

The chatbot can help you with the following topics:

Polynomial Equation Solver

Statistics Calculator

Probability Solver

Trigonometry Calculator

Geometry Calculator

Basic Calculator


It also includes basic conversation ability — so it's not just smart, it's also friendly.It has a dictionary that it can get access from and can also learn from the user (meaning edit the knowledge_base).

---

## ▶️ How to Use

1. Clone or download this repository.
2. Run the script in your Python environment:

```bash
python chatbot.py

3. Say solve math and it will give you main menu and now you can enter 1 for polynomial equation 2 for statistics and so on and so forth. You will only need to enter a number, there might be sub menus.
